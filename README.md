# Frontend Flashcards App

This project is part of a challenge recommended by roadmap.sh, designed to help front-end developers understand state management and build reusable components using modern JavaScript frameworks.

You can find the challenge here: https://roadmap.sh/projects/flash-cards

## Requirements

- Pre-defined flashcards with questions and answers
- Progress bar to show the user’s progress
- Ability to view flashcards one at a time and flip to reveal the answer
- Simple navigation to cycle through the flashcards

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6)
